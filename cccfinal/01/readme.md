# 個人介紹網頁 (Personal Portfolio Website)

這是一個輕量、簡潔且具備響應式設計的個人介紹靜態網頁。主要用於展示個人的簡歷、專業技能以及過往的專案經驗。

## 🌟 專案特色

* **純靜態網頁**：僅使用標準的 HTML5 與 CSS3 打造，無須複雜的框架或環境設定。
* **響應式設計 (RWD)**：支援在電腦、平板與手機等不同尺寸的螢幕上良好閱讀。
* **模組化區塊**：清晰劃分「關於我」、「專業技能」、「專案經驗」與「聯絡方式」等獨立區塊，易於閱讀與維護。

## 🛠️ 使用技術

* **HTML5**：建構網頁基礎語意架構。
* **CSS3**：負責網頁的排版、顏色與視覺樣式（內建於 HTML 檔案的 `<style>` 標籤中）。

## 🚀 如何執行與預覽

1. 將此專案下載或複製到本地端電腦。
2. 確保資料夾中包含 `index.html` 檔案。
3. 直接對著 `index.html` 點擊右鍵，選擇 **「在瀏覽器中開啟」**（如 Chrome, Edge, Safari 等），即可預覽網頁效果。無須安裝任何伺服器環境。

## 📝 如何客製化修改

你可以使用任何文字編輯器（如 VS Code, Notepad++, 或記事本）開啟 `index.html` 進行修改：

1. **替換個人資訊**：搜尋 `[請填入你的名字]` 並替換為你的真實姓名。
2. **修改聯絡方式**：在 `<section id="contact">` 區塊中，更新你的 Email 與 GitHub 連結。
3. **新增專案**：在 `<section id="projects">` 中，複製 `<div class="project">...</div>` 的結構來新增更多你的作品。
4. **調整外觀**：若想更改配色，可以修改 `<style>` 區塊內的色碼（如 `#1abc9c` 為主要綠色，`#2c3e50` 為深藍色背景）。
