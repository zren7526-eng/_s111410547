# 會員註冊表單 (User Registration Form)

## 📝 專案簡介
這是一個使用純 HTML5 與 CSS3 打造的響應式會員註冊表單。專案包含了多種常見的表單控制項，並具備乾淨、現代化的卡片式介面設計，非常適合做為前端網頁開發的基礎模板。

## ✨ 功能特色
本表單整合了以下 8 種常用的 HTML 表單控制項：
* **單行文字 (Text)**：使用者名稱輸入。
* **電子郵件 (Email)**：內建基本的 Email 格式檢查。
* **密碼 (Password)**：隱藏輸入字元，並設定最短 8 碼的字數限制。
* **日期選擇器 (Date)**：呼叫瀏覽器原生彈出式日曆。
* **單選按鈕 (Radio)**：性別單項選擇。
* **下拉式選單 (Select)**：居住地區清單選擇。
* **核取方塊 (Checkbox)**：服務條款同意確認（必填防呆機制）。
* **提交按鈕 (Submit)**：觸發表單資料傳送。

## 🛠️ 使用技術
* **HTML5**: 建立表單語意化結構與內建前端驗證（例如 `required` 與 `minlength`）。
* **CSS3**: 使用 Flexbox 進行畫面完美置中排版，並美化輸入框與按鈕的視覺效果與懸停（Hover）動畫。

## 🚀 如何使用
1. 將本專案的所有程式碼下載或複製到本地端。
2. 確保檔案儲存為 `.html` 格式（例如：`index.html` 或 `register.html`）。
3. 直接使用任何現代的網頁瀏覽器（如 Chrome, Edge, Firefox, Safari）雙擊打開該檔案即可預覽與操作。

> **⚠️ 注意事項：** > 目前表單的 `<form action="...">` 屬性僅為前端介面示範。若要實際傳送與儲存註冊資料，需要自行串接後端伺服器（如 Node.js, Python, PHP 等）或資料庫。
